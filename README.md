# Should a Pakistani Data Professional Target Local or Remote Roles?

**Status:** In progress — data collection & cleaning phase (updated July 7, 2026)

## The question
Pakistani data professionals face a choice: compete locally for Rozee-listed 
Data Analyst/Scientist roles, or upskill toward remote/global roles. This project 
compares skill demand, using real job market data, to answer which path offers 
better ROI and what skills to prioritize either way.

## Data sources
- Stack Overflow Developer Survey 2025 (global skill & tooling trends)
- Kaggle ai-jobs.net dataset (global remote data-role postings)
- Rozee.pk (self-collected Pakistan-specific postings — Data Analyst, 
  Data Scientist, Data Engineer, Business Analyst)
- Rozee.pk postings collected manually (July 2026) after automated scraping was blocked by anti-bot protections — a decision made to preserve project timeline

## Key finding
- In the Pakistan data-analyst market, Excel and SQL are near-universal requirements (35% each),  while big-data tools like Spark and Snowflake appear in under 12% of postings — meaning core fundamentals, not advanced tooling, drive local hireability.
